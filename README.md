# Synthetic Panels Marketing Website

AI-powered focus groups in minutes, not weeks.

## Live Site

https://thecolab-ai.github.io/synthetic-panels-website/

## Features

- Modern dark theme with cyan accent
- Interactive hero demo with typing animation
- Step-by-step walkthrough with live demos
- Counter animations on scroll
- Security visualization
- Responsive design
- Accessible (WCAG 2.1 AA)

## Tech Stack

- HTML5 / CSS3 / Vanilla JS
- Tailwind CSS (CDN)
- Lucide Icons
- Google Fonts (Inter, Space Grotesk, JetBrains Mono)

## Development

Just open `index.html` in a browser. No build step required.

## Deployment

Deployed to GitHub Pages from the `main` branch root.
